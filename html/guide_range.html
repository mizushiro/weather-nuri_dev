TimeRange 신규 기능 테스트 데모

테스트 1: 24시간 이하 범위 (23:00 ~ 다음날 03:00)
기간이 24시간 이하인 케이스입니다. 100% 가로 너비 내에서 전체 4시간 분량이 표시됩니다.
검증 포인트:
1. 슬라이더가 10분 단위로 세밀하게 이동하는지 확인하세요.
2. 슬라이더를 우측으로 움직여 자정(24:00)을 지나 00:00 이후가 되는 순간 상단 날짜 드롭다운이 자동으로 다음날로 변경되는지 확인하세요.
3. 상단 드롭다운에서 날짜를 수동 변경하면 슬라이더 핸들이 해당 날짜의 최초 유효 시간(23:00 또는 00:00)으로 스냅 이동하는지 확인하세요.
이벤트 발생 로그가 여기에 출력됩니다.
테스트 2: 24시간 이상 범위 (시작일 09:00 ~ 종료일 15:00)
기간이 24시간 이상인 케이스입니다. 각 날짜는 무조건 최대 범위인 00:00 ~ 24:00으로 채워집니다.
검증 포인트:
1. 첫날(시작일)은 09:00 이전 구간에 반투명 음영 오버레이가 깔려 선택할 수 없는지 확인하세요.
2. 마지막 날(종료일)은 15:00 이후 구간에 반투명 음영 오버레이가 깔려 선택할 수 없는지 확인하세요.
3. 날짜 드롭다운을 통해 날짜를 변경하면 각 날짜의 눈금선 및 비활성 영역이 올바르게 갱신되는지 확인하세요.
4. 재생 버튼이나 다음 버튼을 통해 하루의 끝에 도달하면 자동으로 다음 날짜로 매끄럽게 넘어가는지 확인하세요.
이벤트 발생 로그가 여기에 출력됩니다.
TimeRange 컴포넌트 개발 가이드

1. 생성자 옵션 (Constructor Options)

옵션명 타입 설명
id String 슬라이더 래퍼 요소의 식별자 (`data-time-range="..."` 값)
rangeType 'datetime' | 'relative' 범위 설정 방식: `'datetime'`(고정 날짜시간) 또는 `'relative'`(현재 시간 기준 상대 기간)
start String `rangeType: 'datetime'`일 때 시작 시간 (`'YYYY-MM-DD HH:MM'`)
end String `rangeType: 'datetime'`일 때 종료 시간 (`'YYYY-MM-DD HH:MM'`)
relativeHours Number `rangeType: 'relative'`일 때 기간 설정 시간 값 (1~240시간, 미설정 시 기본 24)
step Number 분 단위 조절 간격 (기본값 `10`)
playTime Number 자동 재생 시 눈금 이동 시간 간격 (ms, 기본값 `3000`)
onChange Function 값이 변경될 때의 콜백 함수 `(fullDateTimeStr, timeStr, dateObj)`

2. 동적 API 메서드 (Dynamic Methods)

  • init(): 컴포넌트를 초기화하고 이벤트를 바인딩하여 렌더링을 개시합니다.
  • setValue(minutes): 현재 활성 트랙 내에서 슬라이더 값을 설정합니다 (스냅 포인트 자동 보정).
  • setStep(stepMinutes): 조절 간격(`step`)을 실시간 변경하고 트랙 및 핸들을 갱신합니다.
  • setRelativeHours(hours): relative 모드일 때 기간(시간)을 실시간 변경하고 날짜 목록과 슬라이더 트랙을 재구성합니다.
  • updateOptions(opt): 모든 옵션을 동적으로 리셋하고 신규 설정값으로 컴포넌트를 재구성(Destroy & Reinit)합니다.

3. 코드 적용 예시

가. datetime 절대 시간 지정 모드 초기화

const range = new TimeRange({
  id: 'time-range-test1',
  rangeType: 'datetime',
  start: '2026-07-15 23:00',
  end: '2026-07-16 03:00',
  step: 10,
  onChange: (dateStr, timeStr, dateObj) => {
    console.log(`선택 날짜시간: ${dateStr}, 시간: ${timeStr}`);
  }
});
range.init();

나. relative 상대 기간 모드 초기화 및 도중 변경

const range = new TimeRange({
  id: 'time-range-test2',
  rangeType: 'relative',
  relativeHours: 48, // 48시간 지정
  step: 10
});
range.init();

// 도중에 조절 단위(step)를 3시간(180분)으로 변경
range.setStep(180);

// 도중에 상대 기간 범위를 6시간으로 변경
range.setRelativeHours(6);

다. updateOptions를 활용한 전체 설정 리셋

// relative 모드에서 datetime 고정 범위 모드로 완전히 덮어쓰기
range.updateOptions({
  id: 'time-range-test2',
  rangeType: 'datetime',
  start: '2026-07-15 09:00',
  end: '2026-07-17 15:00',
  step: 30
});
